# RI モンテカルロ為替シミュレーター

AWS RDS Reserved Instances (RI) の費用を、モンテカルロ法による為替レートシミュレーションで評価する Web アプリケーションです。

## 概要

円/ドル為替レートの不確実性をモンテカルロシミュレーションでモデル化し、RI（前払いなし・一部前払い・全額前払い）とオンデマンドの年間コストを確率分布として比較します。最悪ケース・期待値・最良ケースを可視化することで、為替リスクを考慮した RI 購入判断を支援します。

## 主な機能

- **為替モンテカルロシミュレーション**: 幾何ブラウン運動モデルで複数の為替パスを生成
- **RI プラン比較**: オンデマンド / RI前払いなし / RI一部前払い / RI全額前払いの年間コスト分布を並列表示
- **コスト分布グラフ**: 各プランの累積コストを中央値・信頼区間付きで折れ線グラフ表示
- **パラメータ調整**: 初期レート・年率ドリフト・ボラティリティ・シミュレーション回数をリアルタイム調整
- **統計サマリー**: 最小・中央値・最大・標準偏差を一覧表示

## 技術スタック

| カテゴリ | 技術 |
|---|---|
| フレームワーク | Vue 3 + TypeScript |
| ビルドツール | Vite |
| スタイリング | Tailwind CSS |
| グラフ描画 | Chart.js + vue-chartjs |
| ユニットテスト | Vitest |
| E2E テスト | Playwright |

## 起動手順

```bash
# 依存関係インストール
npm install

# 開発サーバー起動（http://localhost:5173）
npm run dev

# プロダクションビルド
npm run build

# ビルド結果プレビュー
npm run preview
```

## テスト実行手順

```bash
# ユニットテスト（Vitest）
npm run test

# ユニットテスト（ウォッチモード）
npm run test:watch

# E2E テスト（Playwright）
npm run test:e2e
```
